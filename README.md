Book Notes

Book Notes is a web application that helps users organize and manage their personal reading list.

Features

* Search for books using an external library API
* Add books to your personal collection
* Delete books from the collection
* Reorder books using drag-and-drop functionality
* Open a book page in the library with the Read button
* Dynamic theme switching based on user interaction

Technologies

* Node.js
* Express.js
* PostgreSQL
* EJS
* JavaScript
* HTML & CSS

How It Works

Users can search for books, save them to their collection, and organize them by dragging and dropping items into any order they prefer. Each book includes a Read button that redirects users to the corresponding page provided by the library API.

The application also supports dynamic theme changes, creating a more engaging and personalized user experience.

Installation

npm install
npm start
